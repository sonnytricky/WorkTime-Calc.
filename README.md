# ⏱️ WorkTime-Calc

**WorkTime-Calc** ist ein einfacher, browserbasierter Arbeitszeit-Rechner.
Er hilft dir dabei, deine Pausen zu erfassen und automatisch deinen Feierabend zu berechnen.

Das Tool läuft komplett im Browser und kann direkt über **GitHub Pages** genutzt werden.

---

## 🌐 Live Demo

```
https://DEIN-USERNAME.github.io/REPOSITORY-NAME/
```

---

## ✨ Funktionen

### 🕒 Pausenberechnung

Du kannst drei verschiedene Pausen erfassen:

* Vormittagspause
* Mittagspause
* Nachmittagspause

Für jede Pause gibst du Start- und Endzeit ein, und das Tool berechnet automatisch die Dauer in Minuten.

---

### 🧮 Automatische Feierabend-Berechnung

Das Tool berechnet deinen **Feierabend** basierend auf:

* Start des Arbeitstags
* fixer Sollarbeitszeit (z. B. 08:30 Stunden) <- Sofern in den Einstellungen eingetragen
* eingegebenen Pausen

👉 Ergebnis: exakte Uhrzeit deines Arbeitsendes

---

### ⚙️ Einstellungen

Im Einstellungsmenü kannst du:

* Startzeit des Arbeitstages festlegen
* Sollarbeitszeit definieren
* festlegen, ob bestimmte Pausen in die Arbeitszeit eingerechnet werden sollen:

  * Vormittagspause
  * Mittagspause
  * Nachmittagspause

* Einstellungen zurücksetzen

---

### 📱 Mobile optimiert

Das Tool ist für Smartphone und Desktop optimiert und funktioniert besonders gut auf iPhones als Web-App.

---

## 🚀 Verwendung

1. Öffne die Seite über GitHub Pages
2. Gehe ins Hauptmenü
3. Gib deine Zeiten ein:

   * Arbeitsstart
   * Pausenzeiten
4. Klicke auf **„Berechnen“**
5. Dein Feierabend wird automatisch angezeigt
